# Eevee

A dark Vim color scheme

## Screenshot

![Screenshot](https://i.ibb.co/Rg0G6wT/screenshot.png)

## Installation

### [(vim-plug)](https://github.com/junegunn/vim-plug)

- Add to your `.vimrc`:

```vim
Plug 'jazielloureiro/Eevee-Vim'
```

- And then, run the commands:

```vim
:source %
:PlugInstall
```
